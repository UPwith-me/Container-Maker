# Demo: Modern Web Development with Next.js

This demo showcases Container-Maker for modern web development using Next.js 14.

## Quick Start

```bash
# Initialize the dev container
cm shell

# Install dependencies (inside container)
npm install

# Start development server
npm run dev
```

## Features Demonstrated

- **Node.js 20 LTS**: Latest stable Node.js environment
- **Hot Reload**: Changes reflect instantly
- **TypeScript**: Full type safety
- **Tailwind CSS**: Modern styling
